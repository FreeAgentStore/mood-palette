# Mood Palette

Upload a photo or describe a mood — get a color palette with names. Extract colors from images or generate from text. Pure JS + Chrome Built-in AI.

Part of [FreeAgentStore](https://freeagentstore.online) — AI tools that run free in your browser.

## Install

```bash
npm install @freeagentstore/mood-palette
```

## Usage

```ts
export { extractColorsFromImage, generateFromText, heuristicPalette, colorName } from "@freeagentstore/mood-palette";
```

## ESM (zero install)

```html
<script type="module">
  import { ... } from 'https://freeagentstore.online/pkg/mood-palette/index.js'
</script>
```
